# AlgoMentor AI - Product Requirement Document

## Vision

The Duolingo of DSA inside VS Code.

## Problem Statement

Students learning Data Structures and Algorithms often struggle to understand why a solution works, what its time and space complexity are, which algorithmic pattern it uses, and how to improve it. Existing tools focus on writing code rather than teaching algorithmic thinking.

AlgoMentor AI solves this by acting as an AI-powered DSA mentor directly inside VS Code.

## Target Users

### Primary Users

* College Students
* DSA Learners
* Interview Preparation Candidates

### Secondary Users

* Competitive Programmers
* Coding Bootcamp Students

## V0.1 Features

1. Time Complexity Analysis
2. Space Complexity Analysis
3. Complexity Explanation

Supported Languages:

- C
- C++
- Java
- Python
- JavaScript

## Non Goals

* AI Tutor
* Interview Coach
* Visualizer
* Community Features

## Success Metrics

* Extension Published
* First 100 Installs
* First User Feedback

## Unique Selling Proposition

AlgoMentor AI is the first AI-powered DSA mentor inside VS Code that combines:

- Complexity Analysis
- DSA Pattern Detection
- AI Tutoring
- Test Case Generation
- Interview Coaching

into a single learning-focused workflow.
