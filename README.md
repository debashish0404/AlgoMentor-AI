# AlgoMentor-AI
AI-powered DSA Mentor for VS Code. Analyze complexity, detect algorithmic patterns, generate test cases, and learn Data Structures &amp; Algorithms directly inside your editor.

## Vision

AlgoMentor AI is an AI-powered DSA mentor built directly into Visual Studio Code.

Instead of simply generating code, AlgoMentor helps students understand algorithms, analyze complexity, detect patterns, generate test cases, and prepare for technical interviews.

---

## Mission

Help developers master Data Structures and Algorithms while writing code.
